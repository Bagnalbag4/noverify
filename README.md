## 🌟 Why NoVerify?

Selling digital goods shouldn't require a complex setup or an expensive monthly SaaS subscription. **NoVerify** is a lightning-fast, beautifully designed, completely free React/Vite application tailor-made specifically for sellers of digital accounts, OTP numbers, API keys, and software licenses.

It comes pre-configured with a jaw-dropping **Glassmorphism UI**, a **Live Customer Portal**, a **Support Ticket System**, and a **6-Tab Admin Powerhouse Dashboard**.

## ✨ Key Features

### 🛍️ Frontend (What Your Customers See)
- **High-Converting Landing Page**: Smooth scroll animations, built-in trust badges, and dynamic product showcases.
- **Advanced Checkout Flow**: Multi-quantity support, real-time total calculation, dynamic Payment Gateways (Card, SadaPay, NayaPay, Bank Transfer) and Promo Code integration.
- **Customer Portal**: Authenticated users get their own private dashboard to view past orders and lifetime spend.
- **Support Ticket System**: Direct lines of communication. Customers can open tickets directly from their dashboard.
- **Native Order Tracking**: Non-authenticated customers can check their order status (Pending → Processing → Delivered) natively via `/track` without asking you on WhatsApp.
- **PWA Ready**: fully installable mobile app experience with offline capability.
- **Dark & Light Modes**: flawless UI transitioning.

### 👑 Backend (What You See - Admin CRM)
- **6-Tab Powerhouse Dashboard**: Manage everything seamlessly across Products, Orders, Customers, Analytics, Stock, and Data Tabs.
- **Live Ticket Resolution**: Read and reply to customer support tickets directly inside the Admin Panel.
- **Advanced CRM**: Track customer lifetime value, highlight *VIP* users, and flag *Churn Risks*.
- **One-Click WhatsApp**: Launch WhatsApp chats directly from your dashboard pre-filled with order details.
- **Live Analytics Database**: Powered entirely by real-time Firebase Firestore synchronization.
- **Intelligent Stock Alerts**: Auto-notifies you when a product is running low using native browser alerts and chimes.
- **Data Portability**: 1-click export everything to CSV, or download/upload full JSON state backups.

## 🚀 Quick Start

Get your store running locally in 60 seconds.

### 1. Clone the repository
```bash
git clone https://github.com/yourusername/noverify.git
cd noverify
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Configure your Environment
```bash
cp .env.example .env
```
*Edit `.env` to include your specific WhatsApp number, payment handles, Firebase config, and Admin credentials.*

### 4. Blast Off 🚀
```bash
npm run dev
```
Navigate to `http://localhost:5173`. 
Access your admin panel at `http://localhost:5173/#admin` and login using the credentials you set in your `.env`.
